# 📚 Library Management System

A desktop-based Library Management System built using **Java and Swing** that allows efficient management of books and library transactions through a graphical user interface.

---

## 🚀 Features

- Add new books to the library
- Issue books to users
- Return issued books
- Input validation to prevent invalid operations
- Inventory management using Object-Oriented Programming (OOP)
- User-friendly GUI built with Java Swing

---

## 🛠️ Tech Stack

- Language: Java
- GUI Framework: Java Swing
- Concepts Used: Object-Oriented Programming (OOP), Event Handling

---

## 📂 Project Structure

- `LibraryManagementSystem.java` – Main file containing GUI and core logic

---

## ⚙️ How to Run

1. Clone the repository:
   git clone https://github.com/Chandan9574/Library-Management-System.git

2. Navigate to the project folder:
   cd Library-Management-System

3. Compile the program:
   javac LibraryManagementSystem.java

4. Run the program:
   java LibraryManagementSystem
